Protein Sequence Analyzer
A streamlined Python utility designed to automate the extraction of physicochemical properties from protein sequences. This script provides a high-speed method to parse FASTA files and generate comprehensive reports of key biological metrics.

Key Functionality
Batch Processing: Efficiently handles multiple sequence records from a single input file.
Property Calculation: Computes essential biochemical metrics including:
Molecular Weight: Total mass based on amino acid composition.
Isoelectric Point (pI): The pH level where the protein has a net charge of zero.
Instability Index: A measure of protein stability in experimental conditions.
GRAVY Score: Grand Average of Hydropathicity for hydrophobicity analysis.
Data Export: Generates a structured report (CSV/TXT) for downstream analysis in Excel, R, or Prism.
